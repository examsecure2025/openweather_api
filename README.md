# OpenWeather

## Description

A weather application that provides current weather information for any city worldwide. The application fetches real-time weather data from the OpenWeather API and displays comprehensive weather details including temperature, humidity, wind speed, pressure, visibility, and more. It includes an interactive map showing the exact location of the searched city. The application features both light and dark mode themes for better user experience.

## Features

- Search weather by city name
- Display comprehensive weather information
- Interactive map showing city location
- Weather details including temperature, humidity, wind, pressure, visibility, cloudiness
- Sunrise and sunset times
- Wind direction information
- Dark mode and light mode toggle
- Loading states and error handling
- Responsive design for all devices

## How to Run

1. Open the 'openweather' folder
2. Open 'config.js' file
3. Replace 'YOUR_API_KEY_HERE' with your OpenWeather API key
4. Save the config.js file
5. Double-click on 'index.html' file
6. The weather application will open in your default web browser
7. Enter a city name and click Search

## Files

- index.html - Main HTML structure
- style.css - Styling and layout
- script.js - Weather fetching and display functionality
- config.js - API key configuration

## Usage

- Enter a city name in the search field
- Click Search button or press Enter
- View weather information displayed in a card
- Check the map to see the city location
- Toggle between light and dark mode using the theme button
- All weather details are displayed with clear labels

## API Key Setup

1. Get your free API key from openweathermap.org
2. Open config.js file
3. Replace 'YOUR_API_KEY_HERE' with your actual API key
4. Save the file

## API Source

This application uses the OpenWeather API from openweathermap.org
